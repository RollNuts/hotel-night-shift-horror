# Security And Privacy Policy

This repository is public. Treat every committed file, issue, pull request, screenshot, log, and discussion as publicly visible.

## Never Commit Or Post

- Personal addresses, local machine paths, usernames, account details, or private contact information.
- API tokens, Steamworks credentials, signing keys, certificates, passwords, cookies, or session data.
- Console SDK material or platform-confidential files.
- Paid or restricted raw assets unless redistribution is explicitly allowed.
- Private sales strategy, unpublished contracts, or publisher/platform communications.
- Logs that contain local paths, account names, tokens, private project paths, or crash dumps with personal data.

## Tool And Asset Safety

- Use already installed tools and free official tools only when they do not create charges, collect unexpected data, expose credentials, or violate terms.
- Free assets still require license review before use.
- Do not install plugins, packages, or services that start billing, weaken security, or have unclear license/telemetry behavior.
- Record asset and tool decisions in the appropriate ledger or task before product use.

## If Sensitive Data Is Found

1. Stop work on the affected branch.
2. Remove the data from files.
3. Rewrite public history when necessary.
4. Rotate any exposed credentials immediately.
5. Record the incident privately; do not post the secret in an issue.

For this project, preventing exposure is more important than preserving a neat commit history.
